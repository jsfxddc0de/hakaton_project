# 💃 Школьный Танцевальный Бал 2026

Интерактивное веб-приложение для организации и управления школьным танцевальным балом. Проект состоит из раздельного фронтенда на **React (Vite)** и бэкенда на **FastAPI (Python)** с базой данных **SQLite**. Полностью готов к запуску в контейнерах Docker.

---

## 🚀 Быстрый запуск в Docker

Чтобы запустить проект локально со всеми зависимостями, выполните следующие шаги:

### 1. Клонирование репозитория
```bash
git clone https://github.com/jsfxddc0de/hakaton_project.git
cd hakaton_project
```

### 2. Запуск контейнеров
Запустите Docker Compose (сборка фронтенда и бэкенда произойдет автоматически):
```bash
docker compose up --build -d
```

### 3. Доступ к приложению
*   **Фронтенд (React Web App):** [http://localhost:3000](http://localhost:3000)
*   **Интерактивная документация API (Swagger UI):** [http://localhost:8000/api/docs](http://localhost:8000/api/docs)
*   **Бэкенд API:** [http://localhost:8000](http://localhost:8000)

## 📝 Регистрация новых участников и верификация (Email OTP)

Вы можете зарегистрировать нового участника в реальном времени:
1. Перейдите на вкладку **«Зарегистрироваться»**.
2. Заполните форму. После отправки откроется окно ввода 6-значного кода подтверждения.
3. Поскольку SMTP-сервер настроен для локальной отладки, одноразовый код отправляется в консоль контейнера. Получить его можно командой:
   ```bash
   docker compose logs backend
   ```
4. Найдите в логах строку вида:
   ```text
   [DEBUG VERIFICATION CODE] Email: <ваш-email> | Code: <6-значный-код>
   ```
5. Введите этот код на фронтенде для успешного завершения регистрации и входа в Личный кабинет.
